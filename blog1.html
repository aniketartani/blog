JAVA String length()

by Aniket Artani


String class provides an inbuilt method to determine the length of the Java String. The length() method is used to find the length of the string or the number of characters in the string.

Program


package codesdope;
import java.util.*;

class codesdope
{
public static void main(String[] args)
{
String s1="hello world";
String s2="how are you";
System.out.println(s1.length());
System.out.println(s2.length());
}
}


In the first case, the length of the string "hello world" is 11 because there are 5 characters in hello,1 space, and 5 characters in the world(5+1+5=11). In the second case, the length of the string "how are you" is 11 because there are 3 characters in the words how, are and you, and 2 spaces(3+1+3+1+3=11).
